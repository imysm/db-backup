# 安全配置

本文档介绍如何在 Web 控制台中配置安全功能。

## 加密备份

### 启用加密

1. 进入「任务管理」→ 选择任务 → 「编辑」
2. 展开「存储配置」→「加密设置」
3. 启用加密，选择加密类型
4. 配置加密密钥

### 密钥配置方式

**方式一：直接输入密钥**

```
加密密钥：0123456789abcdef0123456789abcdef0123456789abcdef0123456789abcdef
```

**方式二：从环境变量读取**

```
密钥环境变量：DB_BACKUP_ENCRYPTION_KEY
```

然后在系统环境变量中设置：
```bash
export DB_BACKUP_ENCRYPTION_KEY="0123456789abcdef..."
```

### 生成密钥

```bash
# Linux/Mac 生成 32 字节随机密钥
openssl rand -hex 32
```

### 验证密钥

创建加密任务后，系统会自动验证密钥格式是否正确。

## 登录安全

### 登录限速

系统自动启用登录限速：
- 5 次连续失败后锁定 15 分钟
- 成功后重置计数

### 账户锁定

当账户被锁定时：
- 显示剩余解锁时间
- 等待锁定自动解除

## 数据恢复安全

### 恢复确认

执行数据恢复前：
1. 必须二次确认目标数据库
2. 可选择是否覆盖已有数据（SQL Server REPLACE 选项）

### 加密备份恢复

恢复加密备份时：
1. 系统自动检测加密状态
2. 输入正确的加密密钥
3. 系统解密后执行恢复

## 存储安全

### COS 签名 URL

使用腾讯云 COS 时：
- 下载链接默认使用签名 URL
- 可设置链接有效期
- 无密钥无法访问存储

### 访问凭证

存储凭证配置在环境变量中：
```bash
export TENCENT_SECRET_ID="your-secret-id"
export TENCENT_SECRET_KEY="your-secret-key"
```
